# Dreo Bootstrap Endpoint

**Status:** Verified from official Dreo iOS app traffic.

## Endpoint

GET https://www.dreo-tech.com/access/endpoint

## Query

-   registerTimestamp
-   timestamp
-   lang

## Response

Returns region-specific service endpoints.

## EU response fields

-   appapi: https://app-api-eu.dreo-tech.com
-   deviceapi: https://device-api-eu.dreo-cloud.com
-   websocket: wss://wsb-eu.dreo.com
-   h5: https://fe-eu.dreo.com
-   aiapi: https://aiot-api-eu.dreo-tech.com
-   h5app: https://h5-eu.dreo.com
-   region: EU
