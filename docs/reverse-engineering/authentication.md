# Authentication

## Supported login methods observed in Dreo app

-   Email/password login
-   Apple login
-   Google login

## Current SDK status

The SDK currently supports email/password login only.

## Adapter implication

The ioBroker adapter should eventually support all login methods offered by Dreo.

For the first implementation, email/password login is sufficient for development and testing.

Apple and Google login are future milestones because owner accounts may use social login only and may not have a Dreo password.
